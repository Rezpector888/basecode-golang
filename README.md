## Deployment Development

To deploy API  run

```bash
  go tool air
```

To deploy GRPC run

```bash
  go tool air -c .air.grpc.toml
```

To deploy Worker run

```bash
  go tool air -c .air.worker.toml
```

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
